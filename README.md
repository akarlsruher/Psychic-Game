# Train Schedule App

## About
A simple "guess the letter" game.  The computer chooses a letter, and the user must guess using the keyboard.
## Deployed
https://traintimes.herokuapp.com/
## Technologies Used
HTML, CSS, Boostrap, MomentJS, Javascript, JQuery & Firebase
